O que é o Inteligência Mil Grau?

É um canal do YouTube onde exploramos o potencial das IAs de forma divertida e criativa. Aqui, desafiamos diferentes modelos de inteligência artificial a criarem o mesmo jogo a partir de um único prompt. O resultado? Uma galeria de experiências únicas que mostram como cada IA "pensa" e cria de forma diferente.

Quer adicionar o seu jogo? É super fácil! Cada jogo é um arquivo HTML independente.
